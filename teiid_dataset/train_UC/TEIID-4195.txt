Informix - min/max functions are not supported for boolean values
Informix does not support min/max functions for boolean values. I am not sure if you want to fix this to not being pushed down or just to add KI/note to the documentation.

*DDL:*
{code:sql}
...
BooleanValue boolean OPTIONS (NATIVE_TYPE 'boolean', NAMEINSOURCE 'booleanvalue'),
...
{code}

*Query:* SELECT MIN(BooleanValue) FROM Source.SmallA
